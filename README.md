# The Brownie Club 🍫

Bangalore's finest artisan brownie brand — handcrafted with Belgian chocolate, baked fresh daily.

## Stack

- **Frontend**: Next.js 15 + Tailwind CSS
- **Backend**: Next.js API Routes
- **Database**: MongoDB (Mongoose)
- **Hosting**: Vercel

## Setup

1. Clone the repo
2. `npm install`
3. Copy `.env.example` to `.env.local` and fill in your MongoDB URI
4. `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import in Vercel dashboard
3. Add `MONGODB_URI` environment variable
4. Deploy

## Pages

- `/` — Landing page
- `/menu` — Full brownie menu
- `/order` — Online order form
- `/about` — Brand story
- `/contact` — Contact form + info
